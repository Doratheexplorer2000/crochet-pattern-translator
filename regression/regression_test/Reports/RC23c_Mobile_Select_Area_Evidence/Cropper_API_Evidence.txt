Supported by library:
- box_color
- stroke_width
- realtime_update
- default_coords
- aspect_ratio
- return_type
- should_resize_image

Not exposed by library:
- resize handle size
- resize handle color
- separate mobile touch target size
- floating confirmation controls

Engineering decision:
- Keep realtime_update=True for selection correctness.
- Use supported stroke_width=5 for modest border visibility improvement.
- Do not implement handle-size changes because the current wrapper does not expose a supported option.
- Do not use CSS/JavaScript hacks.
